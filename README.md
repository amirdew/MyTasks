# MyTasks
Simple and easy Task Manager app for iOS.
<br>Written with Xcode(8.3.3) and Swift(3.1)

# How to run
Download or clone project then run this command on terminal:

```ruby
pod install
```
open "MyTasks.xcworkspace"
